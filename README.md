A teensy tiny TypeScript package template. 
## Features
- TypeScript 4.8+
- native ESM support
- Rollup/unbuild - bundless bundler
- jest/ts-jest
- eslint/prettier/git hooks
- CI/CD via GH Actions
## Getting started

```sh

git clone <repository url>
pnpm install

```
